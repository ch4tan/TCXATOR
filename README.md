# TCXATOR

**TCXATOR** is a lightweight web application designed to manually generate TCX files, allowing you to easily add missing activities to your connected watch or fitness platform.

## Features

* **Flexible Export:** Generate a single TCX file or batch-create multiple files packed inside a `.zip` archive.
* **Compatibility:** Fully tested and verified on **Coros** (and compatible with any platform supporting standard TCX imports like Strava, Garmin, etc.).
